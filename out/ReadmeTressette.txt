== tressette a 4 giocatori
Il programma tre.exe é una simulazione del gioco del tressette a 4 giocatori.

== Versione 0.70
Corretto bug dichiarazione 4 assi
Corretto apertura file guida in windows 7
Progetto per visual studio 8
Setup msi

== Versione 0.69
Corretto bug scritta con la vittoria
Accusi con scritte più grandi
Chitarrella cambiato Liscio con Striscio
Help aggiornato

== Versione 0.68
Corretto bug dichiarazione liscio lungo nella variante Chitarella
Aggiornato l'help e corretti errori di ortografia

== Versione 0.66
Aggiunta variante Chitarella. Corretto bug sulle carte napoletane.
Debug game di default (trace_coregame.txt).

== Versione 0.62
Corretto bug algoritmo alpha-beta che ignorava la best line calcolata

== Versione 0.60 beta
Versione beta con algoritmo di gioco migliore (alpha beta a tempo fisso), ma non 
ancora ottimale.
Risolto bug di visualizzazione ordine delle cate.
Risolto bug animazione gioco della carta.
Aggiunte immagini delle opere di Dalcò nello sfondo delle opzioni.  
Settings non vengono più salvati nel registro, ma in un file xml. Parser usato expat.
Aggiunti make files per la compilazione in ambiente linux e freebsd.

Prima versione alpha pubblicata. L'algoritmo di gioco implementato è casuale.
